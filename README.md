# 2026 Local Knowledge Engine (RAG)
**Author:** Harshith Reddy Gundra
**Academic Context:** Masters in CS - Advanced Database Systems

## Tech Stack
- **Orchestration:** Mage AI
- **Database:** DuckDB (VSS Extension)
- **Transformation:** dbt-duckdb
- **Models:** sentence-transformers (Embeddings), Ollama (LLM)

## Setup
1. Place PDFs in `data/raw/`
2. Run `docker-compose up`
3. Access Mage at http://localhost:6789
